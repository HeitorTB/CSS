CSS Inline

O css inline serve para curtas modificações visuais no seu site, sendo mais especifico e alterando pequenos pontos

Exemplo: Para mudar todas as tags h2 para uma determinada cor, seria feito de um em um

Ao lado da tag colocamos o style='' e alteramos o que for necessário

Nunca utilizar

Estilos Locais/Internos

Colocar dentro de head o style e dentro dele, iserir suas modificações

Sendo aplicado para todos os, por exemplo, h2

        
            style

                body {
                    background-color: lightgrey;
                    font-family: Arial, Helvetica, sans-serif;
                    font-size: 15px; 
                }

                h2  {
                    color: darkblue;
                    background-color: lightblue;
                }

            /style

Desvantagens:

  1. Os styles ocupam muito espaço
  2. Não serve para 2 ou mais documentos(páginas)

Harmonia das cores


Tipos de cores

Exemplo: Amarelo-

Cores primárias: Vermelho, azul e amarelo

Cores secundárias: Laranja, violeta e verde

Ambas formam um triângulo equilátero no C.C

Cor terceária: Junção de uma cor P. + cor S.

esverdeado

Formam um hexágono no C.C

O que está entre a esquerda do roxo e a esquerda do verde amarelado são chamadas de cores quentes

Consequentemente, o que se encontra a direita são as cores frias

Geralmente uma paleta tem de 3 a 5 cores


Cores Complementares e Análogas

Cores Complementares: O extremo contrate de uma cor é a que está posicionada opostamente no C.C

Cores Análogas: Cores que tem uma harmonia entre elas

São as 2 cores vizinhas a cor principal (uma à direita e uma à esquerda)

Criar uma paleta: Selecionar a cor principal, pegar as duas análogas a ela e a sua complementar

Tendo assim uma paleta de cores com 4 cores (ideal)

Cores Análogas Relacionadas: Duas cores principais vizinhas + Uma cor depois da vizinha de uma das principais

Forma uma paleta com menos contrate, mais suave

Cores intercaladas: Escolher uma principal, pular uma cor vizinha e pegar a outra 2 vezes, para qualquer direção

Não é muito indicado

Cores Triádicas:Selecionar uma principal e pular 3 para cada lado, formando um triângulo.

Cores em quadrado:Formando uma cruz/quadrado

Cores Tetrádicas: Forma um retângulo no cículo cromático

Monocromia: Pegar uma cor principal e modificar a saturação e a luminosidade através do HSL 3 vezes, formando uma paleta

Utilizar o Adobe Color/Coolors/Paletton(exemplo de site com cor escolhida)

Imagens

Color -> image -> position -> repeat -> **size** -> attachment

Para colocar imagem de fundo, usar a tag: background-image: url("")

Existe a função background repeat, que permite que uma pequena imagem seja multiplicada de acordo com o tamanho da tela

repeat x: eixo x (horizontalmente) / repeat y: eixo y (verticalmente)

Existe também a função: no repeat

background-attachment: fixed + background-size: cover = fixar o plano de fundo e tornar responsivo o conteúdo

-----------------

Se a imagem for muito grande, utilizar o background-position para ajustar

Serve também para a responsividade, ja que ao ampliar ou reduzir o tamanho da tela, a imagem se adaptará de acordo com a position idicada

background-size: cover Serve para mostrar 100% da tela com pequenos cortes da img

background-size: auto Serve para que o navegador se vire para adaptar

background-size: contain Mostrar 100% da tela sem cortes

Tabelas

A tag principal de abetura de tudo de dentro da tabela é table

O tr representa os elementos (representados por td ) dentro da linha horizontal

Com text align center e vertical align middle, consegue centralizar o conteudo da tabela

---------------------

No css colocar a borda (border) e a função: border colapse: colapse, para unir as linhas

As tabelas são divididas em: thead, tbody e tfoot (conclusão)

TR: Cada linha horizontal de assunto, TD: Os elementos dentro do tr, TH: elementos títulos

---------------------

Caption: Título da tabela

th scope row: elementos referidos estão na mesma linha horizontal (so utilizar em th)

th scope col: elementos referidos estão na mesma coluna vertical (so utilizar em th)

Tem como fazer um efeito zebrado na tabela: tbody > tr:nth-child(2n){determinado background color

2n: de duas em 2 linhas

---------------------

colspan: mesclar colunas

rowspan: mesclar linhas

Exemplo: td rowspan='3'

Editar colunas específicas: dentro de table criar um colgroup com varias 'col' dentro, cada uma representando uma coluna da tabela 

            colgroup
                col class="coluna 1">
                col class="coluna 2">
                col class="coluna 3">
            /colgroup

Responsividade: Colocar toda a tabela dentro de um div e no css pôr dentro da div o overflow-x: auto

IFRAME

Tag: iframe

Dentro do src colocar o link da página para que apareça o frame do site

Coloque o link do lado de fora do src também, para que caso nao abra o frame, mostre o link

        iframe src="link da pag" frameborder="0"> link da pag <\iframe>

Frameborder='1' (com borda)

Frameborder='0' (sem borda)

-----------------------------

É possível colocar páginas internas do seu site no próprio iframe

Abrir um iframe específico na tela ao clicar no link:

  1. Cria um menu com diferentes páginas internas(link)
  2. Cria um iframe e define um --name--
  3. Coloca o name do iframe no target dos links do menu

Problema: O iframe enquanto não é escolhido fica em branco

Resolução: Colocar no lugar do src do iframe um srcdoc, e escrever algo dentro com html.
